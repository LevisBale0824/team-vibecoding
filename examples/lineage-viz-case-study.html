// 错误： === RUN TestBuildGraphSelfLoop panic: simple: adding self edge gonum.org/v1/gonum/graph/simple .(*DirectedGraph).SetEdge() // AI 修复：添加前置检查 if fromNode.ID() == toNode.ID() { return nil, fmt.Errorf("cycle detected") } g.SetEdge(simple.Edge{F: fromNode, T: toNode}) // TestBuildGraphSelfLoop PASS ✓
AI Error Handling
"gonum 的 simple.DirectedGraph 在自环时会直接 panic 而非返回 error。解决方法：在 SetEdge 前添加 from != to 检查，用 fmt.Errorf 返回友好错误。"
